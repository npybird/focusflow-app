# FocusFlow

FocusFlow is a modern To-do List application developed as a **Final Project** for the course **CN436: iOS Device Application Development**.
The application is designed to help users manage tasks efficiently with an intuitive interface, smart sorting, and real-time progress tracking.

## 📱 Features

- ✅ Add / Edit / Delete Tasks
- 🎯 Task Priority (Low / Medium / High)
- 📅 Due Date Tracking (Today / Tomorrow / Overdue)
- 🔄 Smart Auto Sorting (based on status, priority, and due date)
- 📊 Progress Ring (visual completion tracking)
- 🔍 Filter System (All / Active / Done)
- 💾 Persistent Storage (UserDefaults)
- ✨ Smooth Animations with SwiftUI

---

## 🧠 Concepts & Architecture

- SwiftUI Declarative UI
- MVVM (Model-View-ViewModel) Architecture
- State Management (`@State`, `@ObservedObject`, `@Published`)
- Local Data Persistence using `UserDefaults` + `Codable`
- Dynamic UI Rendering with Conditional Views
- Animation & Transition Handling

---

## 🛠 Tech Stack

- **Language:** Swift

- **Storage:** UserDefaults

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/npybird/focusflow-app.git
```

2. Open the project in Xcode

3. Select an iPhone Simulator

4. Run the project (⌘ + R)
